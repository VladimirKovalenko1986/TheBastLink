Comands VS-code Terminal
№ п/п Посилання Найменування
1 git stash Коли ви ще не робили git add, git commit i git push, ця команда ховає наш код по типу Ctrl X.
2 git checkout main Войти в main
3 git pull —rebase Взяти актуальний стан main собі в main
4 git checkout (назва вашої гілки) Повернутися до вашої гілки
5 git rebase main Забрати актуальальний стан мейну до себе на гілку.
6 git stash apply Повернути ваш незакомічений код.
7 git clone https://посилання_на_репозиторій клонуєте репозиторій
8 git checkout ‘назва-гілки’ Перехід на гілку ‘назва-гілки’
9 git pull origin ‘назва-гілки’ Стягує останні зміни гілки ‘назва-гілки’
10 git checkout -b ‘назва-нової-гілки’ Створює нову гілку
11 git add . та git commit -m “назва-коміта” Записує зміни до локальної гілки
12 git push origin ‘назва-вашої-гілки’ Публікує ваші зміни на гілці
13 git merge ‘назва-гілки’ Залиття коду з гілки ‘назва-гілки’ на поточну гілку
14 git branch -d ‘назва-гілки’ Видалення гілки ‘назва-гілки’
15 git fetch Стягує всі зміни з віддаленого репозиторію
